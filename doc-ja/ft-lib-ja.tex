\subsection{utility}

\DescribeMacro{\ftymd}
\DescribeMacro{\ymd}
\cmd{\ftymd
  \marg{year}
  \oarg{month}
  \oarg{date}
}

年月日を整形、印字する。縦書きにも対応している。
|\indvdldef|の付加情報での利用を想定している。
\bigskip

\DescribeMacro{\ftundef}
\cmd{\ftundef
  \marg{box name}
}

名前に\meta{box name}を持つものを全て未定義とする。
prefix |ft|なしのaliasは設けない。
